# Git

{: .note-title}
> ⚠️ **What if I already know Git?**
> 
> You can skip the rest of this assignment and upload a link to any project you've created in the past on Github.

You can complete the "Fantasy Cow" module **or** "GitHub Hello World" module for this onboarding assignment.

## Fantasy Cow

Create your own breed of cow, and make a pull request to [https://github.com/codeday/fantasy-cow](https://github.com/codeday/fantasy-cow). Submit a link to your PR once all checks pass.

{: .important-title }
> Not sure how to get started?
> 
> Check the projects CONTRIBUTING.md! Most open source projects have a file like this explaining how to get started as a developer for the project. 

## GitHub Hello World

Complete this assignment, and submit the link to your resulting Github page: [https://guides.github.com/activities/hello-world/](https://guides.github.com/activities/hello-world/)

{: .note-title}
> ❓ **What if I need help with Git?**
> 
> 
> You can ask for help in #ask-for-help or schedule a CSE session.
